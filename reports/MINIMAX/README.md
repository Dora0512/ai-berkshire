## 核心结论（截至2026-07-13）

**当前判断：观望，无安全边际**

> 首轮解禁（占总股本48.9%）叠加港股通缺位（WVR架构未纳入），股价三日暴跌约40%，较历史峰值（1,330港元）回撤超80%，但仍较165港元发行价高约35%。暴跌之后MiniMax并未变"便宜"，只是从"极度透支"回到"大致公允"：现价PS约113倍（2025实际收入）、EV/ARR约48.5倍，三情景概率加权期望价仅比现价高约8%，而悲观情景（模型代差/价格战/监管冲击叠加）下跌幅达72%——赔率结构是风险投资式的，不是价值投资式的。
>
> 生意本质：收入3年增23倍、付费用户177万，需求真实；但毛利率仅25.4%（智谱41%），研发费用是收入的3.2倍，本质仍是"烧钱换份额"。护城河：多模态技术全线第一梯队（OpenRouter编程调用份额全球第一），但优势窗口以月计，无结构性壁垒。管理层：闫俊杰技术判断力一流、本分度暂无污点，但团队年轻、经营记录过短。
>
> 对比同赛道智谱（02513.HK）：智谱B端粘性更强、2026Q1 API提价83%仍放量400%（真实定价权），但估值PS约929倍、EV/ARR约395倍，比MiniMax贵一个数量级，且距历史峰值仅回撤45%（杀估值尚未真正发生）。二选一的话MiniMax赔率更好，但两者当前都不具备价值投资安全边际。
>
